Data

The Fire Department Calls for Service is a public dataset from the city of San Francisco that logs the all the responses to calls received by the fire department. The data contains detail information about the time interval of each call logged, therefore multiple records might be found for a call number. Besides the time interval, each record contains call number, incident number, address, unit identifier, call type, and disposition.
